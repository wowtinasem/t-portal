관리자 로그인

공지·사진을 GitHub에 직접 발행합니다. GitHub 접근 토큰을 입력하세요.

토큰 만드는 법 (최초 1회)
  1. GitHub 로그인 → 우상단 프로필 → Settings
  2. 맨 아래 Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
  3. Repository access → Only select repositories → wowtinasem/t-portal 선택
  4. Permissions → Repository permissions → Contents 를 Read and write 로
  5. Generate token → 복사해서 아래에 붙여넣기
※ 토큰은 이 브라우저 세션에만 저장되며 코드/GitHub에 올라가지 않습니다.
